# No-Code AI platform

No-Code / Low-Code AI platform using Flowise or LangFlow on GKE.

## Setup

```bash
# install required tool dependencies via Brewfile
brew bundle

# create and bootstrap GKE cluster
task create-cluster
task bootstrap-flux
```
